Class AuthProtocol<M, SV, C, AS>

To create an AuthProtocol you need 3 things

  1. authProtocolSchema defines settingsSchema: schema for which authProtocolSettings does this authProtocolLogic need for it to run defines sessionVariableSchema: schema for sessionVariables its capable of producing
  2. authProtocolLogic: defines function which run which run hooks like processIncoming, processOutgoing, ensureActive
  3. authProtocolSettings
